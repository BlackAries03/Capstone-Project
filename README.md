# INSTAKILOGRAM

## WHAT SO SPECIAL FOR INSTAKILOGRAM
### 1. Secure
#### We dont allow unknow user to chat or add u to some random group! You can only chat when both you and   the user u following followed back you. Of course, we will send a notification to the following user.
### 2. We allow dynamic avatar for profile picture!!
#### Better then discord that need u to pay for dynamic avatar, user are free to use dynamic avatar in   INSTAKILOGRAM!
### 3. We have better content moderation
#### We dont allow bad words as title and admin have the permission to remove the post once the post is   being reported.
